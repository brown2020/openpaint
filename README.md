# OpenPaint

A free, web-based vector design app built with Next.js, React, and Firebase. Draw shapes, paths, and text on a layered canvas; optionally sign in to save projects to the cloud.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Firebase (optional)

Create a Firebase project and set client env vars (see `AGENTS.md`). Without them, the editor runs in local-only mode.
